# Getting Started with Pages

Pages plugin for Accelade

## Installation

Install via Composer:

```bash
composer require accelade/pages
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag="pages-config"
```

## Basic Usage

After installation, you can start using the plugin immediately.
